# 新型コロナウイルス検査用試験キットの処理方法について

新型コロナウイルスの感染流行に伴いまして、自宅でPCR検査、抗原検査及び抗体検査ができる検査キットが市販されております。

使用済みの検査キット（プラスチックが主材料で針のないもの）をご家庭から排出する場合につきましては、下記のリンクにあります手順に従って処理をお願いいたします。

- [新型コロナウイルス検査用試薬キットの処理方法(PDF)](https://www.vill.otoineppu.hokkaido.jp/kakuka/juuminseikatsu/oshirase/files/COVID19_kit.pdf)
