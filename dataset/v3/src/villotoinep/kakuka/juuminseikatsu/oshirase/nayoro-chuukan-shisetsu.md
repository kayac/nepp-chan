# （お知らせ）『（仮称）名寄地区一般廃棄物中間処理施設建設工事請負契約』の締結について

日頃より、村民の皆様におかれましては、ごみの分別にご協力いただきありがとうございます。

さて、標記の件について、令和5年9月22日に開催されました『令和5年第3回名寄地区衛生施設事務組合議会臨時会』において、『（仮称）名寄地区一般廃棄物中間処理施設建設工事』の請負契約に係る議案が次のとおり可決されましたので、お知らせいたします。

---

## 契約内容

| 項目 | 内容 |
|------|------|
| 工事名 | （仮称）名寄地区一般廃棄物中間処理施設建設工事 |
| 契約額 | 71億2,690万円（税込） |
| 工期 | 令和5年9月25日 〜 令和9年3月12日 |
| 契約先 | 三機工業株式会社（東京都中央区明石町8番1号） |

## 工事概要

- 焼却施設: 30t/日（15t/16時間×2炉）
- 破砕選別施設: 3.6t/5時間

※入札結果（令和5年8月25日執行）は、[名寄地区衛生施設事務組合のホームページ](https://eiseishisetu.jp/publics/index/61/)に掲載されています。
