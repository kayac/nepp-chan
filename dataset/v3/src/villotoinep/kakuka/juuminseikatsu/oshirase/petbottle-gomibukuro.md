# ペットボトル用ごみ袋の変更について（令和7年1月10日収集より開始）

令和6年12月13日現在、ペットボトル用のごみ袋は村指定の専用袋（10枚入り220円、オレンジ色の半透明袋）に入れていただいて収集しておりますが、皆様のご協力のおかげで奇麗に分別して頂けていること、また、令和6年4月1日からペットボトルの水平リサイクルを実施していることを受けまして、**ペットボトルごみ袋を令和7年1月10日の収集日から市販の白色の半透明袋での収集に変更いたします。**

なお、**オレンジ色の半透明袋である村指定の専用袋も引き続きそのままお使いできます。**

この変更により、**市販の白色の半透明袋は『ペットボトルごみ』と『プラスチック製の容器包装ごみ』の2種類で使用していただきますが、ごみを捨てる際は、必ず中身ごとに袋を分けて出してください。**

もし、**一つのごみ袋に『ペットボトルごみ』と『プラスチック製の容器包装ごみ』を分別せずに混ぜて捨てている場合は、収集しません**のでご注意ください。

この内容は『広報「おといねっぷ」2024年12月号』に掲載の他、下記のクリーンライフにてお知らせしております。

- [クリーンライフ（令和6年12月13日発行）(PDF)](https://www.vill.otoineppu.hokkaido.jp/kakuka/juuminseikatsu/oshirase/files/clean_lifeR6.12.13.pdf)
