# 音威子府村し尿収集手数料及び汚泥収集手数料変更のお知らせ

令和5年2月10日付のクリーンライフにて回覧で周知させていただきました『し尿収集手数料等の改定予定について（令和5年4月1日以降）』のし尿収集手数料について、3月3日開会の令和5年第1回名寄地区衛生施設事務組合議会定例会において料金改定議案が提出及び可決されました。

それに伴いまして、令和5年4月1日から名寄地区衛生施設事務組合が徴収するし尿収集手数料が次のとおり変更となりますのでお知らせいたします。

---

## し尿収集手数料

| 期間 | 料金 |
|------|------|
| 令和5年3月31日まで | 20Lにつき94円（税込） |
| 令和5年4月1日以降 | 20Lにつき**123円**（税込） |

※料金は、汲み取り量に応じたし尿収集専用の収入証紙に記載の金額となります（官公署等を除く）。また、領収書の発行はしていませんが、収入証紙を領収書として扱うことは可能です。

**汚泥収集手数料**につきましても、令和5年4月1日以降から上記の**し尿収集手数料と同額の改定**となりますので、ご留意願います。

村民の皆様には多大なご負担をかけますが、ご理解の程、宜しくお願い致します。

後納でお支払いいただいている皆様につきましても、下記のとおりし尿収集手数料変更のお知らせをいたします。ご理解の程、宜しくお願い致します。
