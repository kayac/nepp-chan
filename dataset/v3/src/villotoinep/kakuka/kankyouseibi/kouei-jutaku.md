# 公営住宅の申し込み

村内の公営住宅は、空き室が発生した時に入居の公募を行っています。

入居の申し込みには、定められた申込書（環境整備室にあります）とともに、次の書面を添付してください。

1. 入居申込者と同居している家族全員の住民票
2. 入居申込者と同居しようとする者が婚姻の届出をしていないが、事実上婚姻関係と同様の事情にある者、その他婚姻予約者である者については、その事実を証明する書類
3. 入居申込者および入居しようとする親族の収入に関する金額を証明する書類
4. その他村長が必要と認める書類
