# 水質検査計画および水質検査結果について

水道法施行規則に基づき、安全で快適な水道水が供給できるよう、検査地点や検査項目、検査回数などを定めた水質検査計画を策定しました。

また、水質検査計画と水質検査結果の公表を行うなかで、皆さまからいただいた意見・要望を今後の水質検査計画に反映させ、より安全で信頼できる水道をめざします。
