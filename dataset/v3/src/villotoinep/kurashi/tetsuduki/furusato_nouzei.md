# ふるさと納税制度について

北海道で一番小さな村・音威子府村の村づくりを「応援したい」「何かの形で貢献したい」という思いをお持ちの皆様、ふるさと納税制度を通じ音威子府村の応援団になっていただけませんか。

音威子府村では、ふるさと納税として**5,000円以上の寄附**をいただきました方に、返礼品として村内の特産品などをお送りしています。

なお、下記リンクのポータルサイトでは、数量および期間、サイト限定の返礼品もございますので、ぜひご覧ください。

## ふるさと納税寄附金の使途について

ご寄附をいただきました皆さまに対し、心からお礼申し上げます。
皆さまからいただいたご寄附は、村政の発展のために有効に活用させていただきます。
今後とも音威子府村への応援をどうぞよろしくお願いいたします。
