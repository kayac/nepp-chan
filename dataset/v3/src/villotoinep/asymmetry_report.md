# 情報非対称性レポート: 音威子府村ナレッジベース

## 1. 概要
本プロジェクトでは、音威子府村公式ウェブサイトおよび関連するPDF資料から情報を収集し、Markdown形式のナレッジベース (`knowledge/otoinep/`) を構築しました。
本レポートでは、作成されたテキストベースのMarkdownファイルと、実際のウェブサイト/元ファイル（PDF）との間に存在する情報の非対称性（ギャップ）について報告します。

## 2. 収集・変換の状況
- **ウェブページ**: 主要カテゴリ（行政、くらし、産業、防災、村について）のテキスト情報をMarkdown化しました。
- **PDF資料**: 防災計画、ハザードマップ、入札情報、広報誌などの主要PDF（計20ファイル以上）を `pdftotext` を用いてテキスト抽出し、Markdown形式に変換して保存しました (`pdf/parsed/`)。
  - *注記*: PDF変換時には、ページ番号、ヘッダー/フッター、地図のスケール表記などのノイズ情報を除去するスクリプト処理 (`convert_pdfs.py`) を適用し、検索性と可読性を向上させています。

## 3. 情報非対称性の分析

### 3.1 視覚情報の欠落
最も大きな非対称性は、地図や図表などの視覚情報にあります。
- **ハザードマップ (`map202003.md`)**:
  - **現状**: 避難所のリスト、心得、連絡先などの「テキスト情報」は抽出されています。
  - **欠落**: 実際の地図（浸水想定区域、土砂災害警戒区域の着色マップ）はテキスト化により完全に消失しています。利用者は具体的な危険区域を視覚的に把握できません。
- **ごみカレンダー**:
  - **現状**: 収集日の記述や分別ルールはテキスト化されていますが、カレンダー形式（日付配置）の視認性は再現されていません。

### 3.2 構造化データの崩れ
PDFから抽出されたテキストは、元のレイアウト（段組み、表組み）を完全には維持していません。
- **各種申請書・様式**:
  - **現状**: 入力項目名などはテキストとして保持されていますが、申請書としてのレイアウトや記入欄は失われており、印刷しての利用は不可能です。
- **複雑な表**:
  - **現状**: 予算書や統計データなどは、`pdftotext -layout` オプションによりある程度の配置は保たれていますが、厳密な表構造（マークダウンのテーブル構文）にはなっていません。列位置のズレにより数値を読み間違えるリスクがあります。

### 3.3 施設詳細情報の深さ
- **施設案内 (`shisetsu/`)**:
  - **現状**: 公式サイト上のリスト情報を基に作成しているため、施設名と概要のみにとどまっています。
  - **欠落**: 個別の施設詳細（最新の料金表、営業時間変更、予約状況など）は、個別の詳細ページや外部サイト（予約システム等）に依存する場合があり、現在のナレッジベースには含まれていません。

### 3.4 動的情報の非同期
- **お知らせ・緊急情報**:
  - **現状**: 収集時点（スナップショット）の情報です。公式サイトで更新される新着情報は反映されません。
  - **改善案**: 「お知らせ」や「人口」、「注目のキーワード」などはWebサイト更新頻度が高いため、1日1回の更新確認を推奨します。

## 4. 解決策と推奨事項
この非対称性を補うために、以下の運用を推奨します。

1.  **元ファイル参照の徹底**:
    - Markdownファイル内には常に元PDFへのリンクが含まれています。詳細な確認（特に図面や表）が必要な場合は、必ず元ファイルを参照してください。
2.  **ノイズ除去の継続的改善**:
    - PDF変換スクリプトは改善されましたが、新たなフォーマットのPDFが追加された場合、不要なヘッダー/フッターが残る可能性があります。その際は `convert_pdfs.py` の除外パターンを追加してください。
3.  **重要図版の画像管理**:
    - ハザードマップや観光マップ等は、テキスト化だけでなく画像ファイルとしてナレッジベースに保存することを検討してください。

### 4.2 印刷・視覚利用への誘導（対応済み）
ハザードマップ、ごみカレンダー、申請手引きなど、紙媒体での利用や視覚的な確認が重要となる情報については、各Markdownファイル内に**「※推奨: 印刷・詳細確認用リンク」**等の注記を追記しました。これにより、ユーザーが必要に応じてスムーズに元PDFへアクセスできる導線を確保しています。

## 5. 結論
構築されたナレッジベースは、**「テキスト検索による情報への高速アクセス」**においては非常に有効です。特に複数のPDFにまたがる情報を横断的に検索できる点は大きなメリットです。
一方で、**「視覚的確認」**や**「申請利用」**、**「最新情報の確認」**においては、公式サイトおよびオリジナルPDFによる補完が必須となります。
各ファイルに設置した誘導リンクを活用し、用途に応じてテキスト情報と元ファイルを使い分ける運用を推奨します。
