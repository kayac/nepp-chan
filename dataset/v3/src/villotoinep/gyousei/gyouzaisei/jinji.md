# 人事行政の運営等の状況報告

地方公務員法では、地方公共団体の長は、条例で定めるところにより、毎年、職員の任用、給与、勤務時間その他の勤務条件、分限・懲戒、研修、勤務成績の評定、福祉、利益の保護等、人事行政の運営の条件を公表することとされています。

本村におきましても地方公務員法に基づき、『音威子府村人事行政の運営等の公表に関する条例』を制定し施行していますので、同条例第6条および第7条により公表いたします。
