# 音威子府村の施設・観光スポット・店舗

音威子府村にある主要な施設、観光スポット、店舗の情報です。

## 公共施設・観光スポット

### 音威子府村役場
*   **住所**: 北海道中川郡音威子府村字音威子府444番地1
*   **電話番号**: 01656-5-3311
*   **概要**: 村の行政施設。営業時間は8:30～17:15（土日祝・年末年始休業）。

### 道の駅 おといねっぷ
*   **住所**: 北海道中川郡音威子府村字音威子府155
*   **電話番号**: 01656-5-3111
*   **概要**: 「森と匠の村」をイメージした木造建築。名物「音威子府そば」や木工芸品を販売。食堂「天北龍」併設。営業時間9:30～17:00（冬季～16:00）、火曜定休。

### 音威富士スキー場
*   **住所**: 北海道中川郡音威子府村字音威子府200-6
*   **電話番号**: 01656-5-3305
*   **概要**: JR音威子府駅から徒歩約10分。雪質が良く、初心者から上級者まで楽しめる。営業期間12月中旬～3月下旬。

### エコミュージアムおさしまセンター（アトリエ3モア）
*   **住所**: 北海道中川郡音威子府村字物満内55番地
*   **電話番号**: 01656-5-3980
*   **概要**: 彫刻家・砂澤ビッキのアトリエを改修した美術館。100点以上の作品を展示。営業期間4/26～10/31、月曜休館。

### 天塩川交通ターミナル（天北線資料室）
*   **住所**: 北海道中川郡音威子府村字音威子府509番地40
*   **概要**: JR駅・バスターミナルに併設。廃止された天北線の資料やジオラマを展示。資料室開館9:00～17:00。

### その他施設
*   **高橋昭五郎彫刻の館**: 彫刻家・高橋昭五郎氏の作品展示。
*   **木遊館**: 木工芸体験施設。
*   **天塩川温泉**: 保養センター。
*   **中島公園**: キャンプ場など。
