# （3月7日更新）名寄地区広域最終処分場及び炭化センターへの自己搬入の自粛に関するご協力のお願いについて

日頃より、村民の皆様におかれましては、ごみの分別にご協力いただきありがとうございます。

標記の件について、北海道に1月27日から3月21日までの期間、まん延防止等重点措置の適用が決定されました。

この決定を受けまして、新型コロナウイルスの感染拡大の防止の観点から、場内の混雑緩和、施設従事者との接触応対を避けるために、下記の期間まで、名寄地区広域最終処分場及び炭化センターへの自己搬入を自粛していただきますようご協力をお願いいたします。

**自粛をお願いする期間: 令和4年3月21日まで**

※今後の国・北海道の動向及び感染状況により、期間を変更する場合があります。

今後の情報等は、[名寄地区衛生施設事務組合のホームページ](http://eiseishisetu.jp/)に随時公開していきますので、ご参照ください。

- [（3月7日差替）名寄地区衛生施設事務組合からのご協力のお願い(PDF)](https://www.vill.otoineppu.hokkaido.jp/kakuka/juuminseikatsu/oshirase/files/mochikomi_jisyukuR4.3.7.pdf)

## やむを得ず自己搬入される場合の留意点

- 体調が悪い時は、来場を見合わせてください
- 必要最小限の人数でお越しください
- マスクの着用を必ずお願いいたします
- 係員の指示に従っていただきますようお願いいたします
- 係員と会話をする場合は、一定の距離を保ってください
