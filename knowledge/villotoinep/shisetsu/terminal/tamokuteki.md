# 音威子府村交通ターミナル 多目的スペース

交通ターミナルの活性化を目的とし、2021年より地域住民の交流スペース、ギャラリースペース、イベント利用空間として整備された多目的スペースです。

## 施設概要

| 項目 | 内容 |
|------|------|
| **名称** | 音威子府村交通ターミナル多目的スペース |
| **所在地** | 音威子府村交通ターミナル内（音威子府509番地40） |
| **面積** | 約8.34㎡（間口2.35m×奥行3.55m×高さ2.23m） |
| **管理者** | 音威子府村（総務課地域振興室） |

## 利用対象者

- 管理者から利用許可を得た者
- 個人、団体、住民グループなど（村内居住有無を問わない）

## 利用可能期間

### (a) ギャラリー利用

- 期間: 最低1週間〜最長2か月以内
- 内容: 平面作品、立体作品、映像作品など
- 注意: 壁や床を汚すようなものは対象外

### (b) イベント利用（営利目的でないもの）

- 期間: 最低1日間もしくは短期間（3日程度）
- 飲食物提供等は利用者が保健所への「臨時営業許可」手続きが必要
- **営利目的の使用は不可**（JR北海道との契約上、事業所や個人事業主の販売利用不可）
  - 内容によっては物販可能な場合あり（要相談）

### (c) その他

- 「地域住民の交流機会」が主目的の利用は原則可能
- 上記以外の利用希望は別途相談

## 申請方法

1. 利用申請書に必要事項を記入
2. **総務課地域振興室**に提出
3. 内容確認後、利用可否を連絡

- 申込期限: 利用開始日の**1週間程度前まで**
- おといねっぷ美術工芸高校の生徒は、先生の署名・押印が必要
