# 国民健康保険について

国民健康保険（国保）は、病気やケガをしたときに安心して医療を受けられるよう、加入者がお金を出し合い、医療費の負担を軽減する相互扶助の制度です。

---

## 国民健康保険の加入・脱退

### 加入するとき

| ケース | 届出に必要なもの |
|--------|------------------|
| 他の市町村から転入したとき | 印鑑、本人確認書類 |
| 職場の健康保険をやめたとき | 健康保険資格喪失証明書、印鑑 |
| 子どもが生まれたとき | 印鑑、母子健康手帳 |
| 生活保護を受けなくなったとき | 保護廃止決定通知書、印鑑 |

### 脱退するとき

| ケース | 届出に必要なもの |
|--------|------------------|
| 他の市町村へ転出するとき | 国保保険証、印鑑 |
| 職場の健康保険に加入したとき | 国保保険証、新しい保険証、印鑑 |
| 加入者が死亡したとき | 国保保険証、印鑑 |
| 生活保護を受けるようになったとき | 国保保険証、保護開始決定通知書、印鑑 |

---

## 国民健康保険で受けられる給付

| 給付の種類 | 内容 |
|-----------|------|
| 療養の給付 | 医療機関での診療費の一部を負担（通常3割） |
| 高額療養費 | 1ヶ月の医療費が高額になったときの払い戻し |
| 出産育児一時金 | 出産したときに支給（1児につき50万円） |
| 葬祭費 | 加入者が死亡したときに支給（3万円） |
| 療養費 | やむを得ず保険証を持たずに受診したときの払い戻し |

---

## 高額療養費制度

1ヶ月の医療費が自己負担限度額を超えた場合、超えた分が払い戻されます。

### 自己負担限度額（70歳未満の場合）

| 所得区分 | 自己負担限度額（月額） |
|---------|----------------------|
| 年収約1,160万円〜 | 252,600円＋（医療費−842,000円）×1% |
| 年収約770〜1,160万円 | 167,400円＋（医療費−558,000円）×1% |
| 年収約370〜770万円 | 80,100円＋（医療費−267,000円）×1% |
| 〜年収約370万円 | 57,600円 |
| 住民税非課税 | 35,400円 |

### 限度額適用認定証

事前に「限度額適用認定証」の交付を受けると、医療機関窓口での支払いが自己負担限度額までとなります。

---

## 保険証を持たずに受診したとき

急病などでやむを得ず保険証を持たずに受診した場合、いったん全額を支払い、後日申請することで自己負担分を除いた額が払い戻されます。

**申請に必要なもの**:
- 診療内容明細書
- 領収書
- 国保保険証
- 印鑑
- 振込先口座がわかるもの

---

## 交通事故などにあったとき

交通事故など第三者の行為によりケガをした場合でも国保で治療を受けられますが、届出が必要です。

**届出に必要なもの**:
- 第三者行為による傷病届
- 事故証明書
- 国保保険証
- 印鑑

---

## マイナ保険証について

マイナンバーカードを健康保険証として利用できます。

**メリット**:
- 医療機関での受付がスムーズ
- 過去の処方歴や特定健診情報の共有が可能
- 高額療養費の限度額適用が自動化
