## 🕒 乗り継ぎモデルコース（参考例）

※あくまで一例です。季節・曜日により接続できない場合があります。必ず最新時刻表をご確認ください。

### 【往路】東京 → 音威子府に到着するパターン
**(A) 昼に到着する便を利用（特急宗谷に接続）**
1.  **[空路]** 羽田空港 07:00頃発 → 旭川空港 08:35頃着
2.  **[バス]** 旭川空港 09:00頃発 → 旭川駅 09:40頃着（空港連絡バス）
3.  **[休憩]** 旭川駅周辺で昼食・買い物（約4時間）
    - *※午前中の接続列車特急は無いため、バス利用または待機となります。*
    - *※バスの場合: 「特急えさし号」旭川駅 11:30発 → 音威子府 13:40着 など*
4.  **[鉄道]** 旭川駅 13:35発 → 音威子府駅 15:15着（特急宗谷）

**(B) 午後に到着する便を利用（特急サロベツに接続）**
1.  **[空路]** 羽田空港 12:15頃発 → 旭川空港 13:50頃着
2.  **[バス]** 旭川空港 14:30頃発 → 旭川駅 15:10頃着
3.  **[鉄道]** 旭川駅 20:06発 → 音威子府駅 21:40着（特急サロベツ1号）
    - *※旭川駅での待ち時間が長いため、夕食等をおすすめします。快速なよろ号等を乗り継ぐルートもあります。*

---

### 【復路】音威子府 → 東京へ帰るパターン
**(A) 朝出発・昼帰着（特急サロベツ利用）**
1.  **[鉄道]** 音威子府駅 08:26発 → 旭川駅 09:59着（特急サロベツ2号）
2.  **[バス]** 旭川駅 10:25頃発 → 旭川空港 11:05頃着
3.  **[空路]** 旭川空港 12:00頃発 → 羽田空港 13:50頃着

**(B) 夕方出発・夜帰着（特急宗谷利用）**
1.  **[鉄道]** 音威子府駅 17:52発 → 旭川駅 19:28着（特急宗谷）
2.  **[バス]** 旭川駅 19:40頃発 → 旭川空港 20:20頃着
    - *※最終便への接続は季節や曜日によりシビアなため要注意*
3.  **[空路]** 旭川空港 20:55頃発 → 羽田空港 22:40頃着
